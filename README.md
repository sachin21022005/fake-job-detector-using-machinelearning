
# 🚨 Fake Job Detector (ML Project)

## 📌 Overview

A machine learning-based system that detects whether a job posting is **real or fake** using NLP and classification algorithms.

---

## 🎯 Objective

To prevent job scams by automatically identifying fraudulent job listings from user input.

---

## ⚙️ Tech Stack

* Python
* Scikit-learn
* Pandas, NumPy
* NLTK
* Flask (for web app)

---

## 🧠 Model

* TF-IDF Vectorization
* Logistic Regression / Naive Bayes

---

## 🚀 How to Run

```bash
git clone https://github.com/your-username/fake-job-detector.git
cd fake-job-detector
pip install -r requirements.txt
python app.py
```

---

## 📊 Output

* ✅ Real Job
* ❌ Fake Job
